# MEMBRA Protocol

Membra_api is the system of record.

This repo follows the shared Membra protocol for QR routing, NFC tap tracking, campaign attribution, placement lookup, and analytics handoff.

Core rule: certified campaign codes route through the gateway first so scans and taps can be measured before final destination routing.

Shared IDs: own_, adv_, ast_, cmp_, plc_, kit_, qr_, nfc_, proof_, scan_, tap_, aud_.
